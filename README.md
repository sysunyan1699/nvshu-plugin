女书插件是一个致力于将女性力量和独特的文化艺术融入您的图片的工具。用途只有一个就是将女书文字嵌入用户钟爱的图片中。在提供此功能时，女书插件仅会读取用户在图片上插入的文字以及监听点击操作，同时不会有任何用户数据被进行网络传输或者后台保存。
